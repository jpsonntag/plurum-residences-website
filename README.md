# Plurum Residences Marketing

Marketing-Website für **Plurum Residences** — den ersten öffentlichen Hub der Marke Plurum.

Ziel: Möblierte WG-Zimmer für Studierende präsentieren und die technische Basis schaffen, um später Wohnungen, Bilder, Verfügbarkeiten und Inhalte redaktionell pflegen zu können.

## Status

Initiales Next.js-Repository mit:

- Startseite für Plurum Residences
- übernommenem Plurum-Logo aus `drax/client-work/plurum-residences`
- vorbereiteter Datenstruktur für Wohnungs-/Zimmer-Einträge
- Projekt- und Continuity-Dokumentation für spätere Agenten-Sessions

## Lokale Entwicklung

```bash
npm install
npm run dev
```

Verification:

```bash
npm run verify
```
