# Onboarding Summary

Dieses Repo wurde als neues Client-Work-Projekt für die Plurum Residences Marketing-Website gestartet.

Die Website dient als öffentlicher Hub für möblierte Studierenden-WGs und ist gleichzeitig als Startpunkt für eine spätere Plurum-Marken-/Holding-Struktur gedacht.

Technisch ist ein kleines Next.js/TypeScript-Projekt vorbereitet. Wohnungsdaten liegen zunächst als statische TypeScript-Daten vor und können später durch CMS/API ersetzt werden.
