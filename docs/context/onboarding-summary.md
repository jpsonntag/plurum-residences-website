# Onboarding Summary

Dieses Repo ist das Client-Work-Projekt für die Plurum Residences Marketing-Website.

Die Website dient als öffentlicher Hub für möblierte Studierenden-WGs und ist gleichzeitig als Startpunkt für eine spätere Plurum-Marken-/Holding-Struktur gedacht.

Technisch ist die öffentliche Marketing-Seite jetzt eine statische Hugo-Site. Residences und Zimmer werden initial manuell über Markdown in `content/residences/` und `content/rooms/` gepflegt. Dynamische Plattformfunktionen können später separat entstehen.
